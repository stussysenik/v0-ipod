# Design Tokens

Source of truth lives in `tailwind.config.ts` and `app/globals.css`. This
page is auto-generated from `design-tokens/tokens.json` — the same file the
Figma sync imports into Variables.

> Editing tokens directly in Figma does **not** round-trip. Token edits flow
> code → Figma one-way (Phase 1) and optionally Figma → code for `Primitives`
> and `Semantic` collections only (Phase 3). See `docs/figma/runbook.md`.
