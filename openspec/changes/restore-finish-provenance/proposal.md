# Change: Restore finish provenance and converge on an anniversary-grade hero artifact

## Why
The project already has breadth, but the main job for this phase is depth: deliver a historically respectful, physically believable, export-ready iPod Classic suitable for an Apple 50 years celebration. The current repo still spreads effort across many surfaces and utilities while the core Flat/Preview object, finish system, picker workflow, and export parity are not yet resolved to tribute quality.

## What Changes
- Reframe this change around a single goal: a definitive Flat/Preview hero artifact
- Treat industrial-design realism, finish provenance, color-selection UX, and export parity as the core workstreams
- Keep 3D, Focus, and ASCII available but explicitly secondary and non-blocking for production quality
- Add a ruthless implementation order so tasks reflect what must land first for anniversary quality

## Impact
- Affected specs: visual-fidelity, finish-provenance, surface-hierarchy
- Affected code: components/ipod/*, lib/* export and theme logic, tests/* regression coverage, docs project positioning
